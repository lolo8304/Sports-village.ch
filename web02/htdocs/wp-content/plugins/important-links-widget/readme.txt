== Description ==

Adds a simple widget that contains important links for use on a community blogging site - new post, new page, login, logout, etc...

== Installation ==

Installing the plugin is as easy as unzipping and uploading the entire important-links-widget folder into your /wp-content/plugins directory and activating the plugin. It can be activated on individual sites, or an all sites under WPMU.

== Basic Usage ==
Install and enable plugin. Add the widget wherever you want it via Appearance: Widgets. That is all.
